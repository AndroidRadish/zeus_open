# T-003 AI Log — 实现 collect_metrics.py

## Decision Rationale

1. **纯 Python 替代 bash 脚本**：collect-metrics.sh 依赖 bash、date、jq 等 Unix 工具，在 Windows 上无法直接运行。选择 Python 3 标准库实现，确保跨平台兼容。
2. **数据源扩展性**：保留了原脚本的 4 种数据源示例（SQLite、PostgreSQL、GA CSV、HTTP API），并用 Python 内置/可选库替代外部命令：
   - sqlite3（内置）
   - csv（内置）
   - urllib.request（内置）
   - psycopg2（可选，失败时友好提示）
3. **CLI 设计**：参考 generate_tests.py 的风格，使用 argparse 提供 `--source`、`--db`、`--dsn`、`--csv`、`--api`、`--token`、`--out`、`--version` 等参数，既支持自动探测也支持显式指定。
4. **编码兼容**：复用了 zeus_runner.py 中的 Windows console encoding fix，确保 emoji 和中文在 PowerShell/CMD 中正常输出。
