## Decision Rationale

The approved design spec defines a Hub-and-Spoke architecture with a unified FastAPI backend serving both a PyQt desktop GUI and a zero-build Web frontend. Planning artifacts break the work into 4 waves: Infrastructure → Core Engine → Dual Frontend → Integration & Ship. Wave boundaries respect the dependency DAG and keep human approval gates at wave transitions.
