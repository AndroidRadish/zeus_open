# T-022 — Update zeus-execute-v2.md with workspace bootstrap documentation

## Decision Rationale

A feature is only shipped when users can discover and configure it. The subagent workspace bootstrap mechanism needed clear documentation in the primary execution reference (`skills/zeus-execute-v2.md`) so that users know:
1. Which files are auto-copied by default.
2. How to customize the file list via `config.json`.
3. That missing files are skipped safely without breaking dispatch.

- Modified `skills/zeus-execute-v2.md`:

  - Listed the four default identity/context files.
  - Documented the optional `subagent.bootstrap.files` configuration with a full JSON snippet.
  - Clarified that the override replaces—not merges with—the default list.
