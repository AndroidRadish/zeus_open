# T-025 — Seed i18n data for all v2 tasks and update documentation

## Decision Rationale

Schema and UI support for i18n is worthless without actual data. To make the feature immediately useful, every existing v2 task needed a Chinese title translation (and English description where present). Automating the seed via a script ensured consistency and avoided manual copy-paste errors across 25 tasks.

## Execution Summary

- Wrote `seed_i18n.py` to bulk-populate `.zeus/v2/task.json`:
  - `title_en` = existing English title for all tasks.
  - `title_zh` = concise Chinese translation tailored to each task.
  - `description_en` = existing English description (T-020 ~ T-025).
  - `description_zh` = corresponding Chinese translation.
- Removed the one-off script after execution to keep the repo clean.
- Updated `skills/zeus-execute-v2.md` with a "Task Multi-language Support" section documenting field names, fallback rules, and orchestrator behavior.
- Commit: `b5dd912`

## Target Impact

- `developer_adoption_rate` ↑: users can immediately experience the language toggle because all visible tasks are bilingual.
- Reduces future maintenance burden by establishing a clear pattern for adding i18n to new tasks.
