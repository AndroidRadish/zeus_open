## Decision Rationale

用户对纯 Python 原生 SVG 生成器的视觉效果不满意，并决定彻底放弃 PyQt 桌面 GUI，将 v2 前端统一为零构建 Web UI。根本原因：

1. **PyQt 的维护成本高**：需要额外安装 `PyQt6`，在不同平台上样式差异大，且无法轻松达到 Web UI 的视觉效果。
2. **Web UI 已经足够好**：基于 Vue 3 + Tailwind CSS + Mermaid 的 `index.html` 无需 npm/build，跨平台一致，暗色玻璃拟态主题已经成熟。
3. **SVG fallback 的问题被绕过**：移除 PyQt 后，桌面端用户统一通过浏览器访问 Web UI，而 Web UI 使用 Mermaid 直接在前端渲染图谱，不再需要系统级 Graphviz。

决策：删除 `zeus_gui.py` 及其测试文件，将所有文档和配置统一为仅维护 Web UI。
